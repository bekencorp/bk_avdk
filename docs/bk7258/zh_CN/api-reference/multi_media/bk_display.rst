LCD Display Interface
----------------------------

The BK LCD Display Driver supports following interfaces:

 - i8080 LCD  interface
    1) 项目中使用的屏幕Driving IC 为ST7796S.
    2) 8080 接口的数据传输为8位
    3) 接口支持的数据为RGB565

 - RGB interface
    1) 项目中支持的屏幕有Driving IC ST7282，HX8282, GC9503V.
    2) 接口支持的数据包括RGB565,YUYV,UYVY,YYUV,UVYY,VUYY
